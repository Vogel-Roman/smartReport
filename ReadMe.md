1. Необходимо прочитать файл настроек settings.json и извлечь из него путь до корневой папки BazisMebelschik

по материалам и фурнитуре. 1 Запрос в Базу данных!!!

Далее создаем общее количество по проекту в целом. И вот эта информация собирается в Смету:
По изделиям, Общая

По кадому листовому материалу и погонному создается файлик excel-спецификация для подгрузку в систему контроля производства

Так же создается файл сметы по изделия (вкладки) и сводная смета по Проекту в целом (вкладка)
Из файла настроек берем настройки коэффициентов наценки по каждому классу материалов. Так же из файла настроек берем стоимости раскроя для разных толщин материлов, если в настройках не задан флаг, что считать по площади. В таком случае в смету попадает только площадь деталей материалов соответсвующего класса

Сортировку материалов можно выполнить по полю Description из Базы материалов. Т.е. туда проставить свои наменования и получить срупприванные материалы по этому полю.
